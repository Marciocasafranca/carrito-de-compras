Descripción:
Este proyecto es una página web interactiva que simula un carrito de compras utilizando HTML, CSS y JavaScript.

Se implementaron:
- Uso de JavaScript básico (valores, tipos, operadores, estructuras de control).
- Funciones: normales, flecha, recursivas y crecientes.
- Estructuras de datos: objetos, arrays, clases, prototipos, métodos, encapsulamiento, mapas y polimorfismo.
- Manejo de eventos: click, teclado, carga, scroll, foco y temporizador.
- Diseño responsive y moderno usando CSS3.

Instrucciones:
1. Abre el archivo index.html en tu navegador.
2. Interactúa con los botones para agregar productos al carrito.
3. Presiona Enter para finalizar la compra.
4. Vacía el carrito con el botón correspondiente.
